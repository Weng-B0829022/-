\chapter{緒論}

\section{研究背景}

複雜性科學作為二十一世紀跨領域研究的核心領域之一，已成為理解自然與人造系統中秩序與混沌共存現象的重要理論基礎。從生物的基因調控、神經訊號傳導，到社會互動、資訊傳播與交通運輸等系統，皆能以節點與連結構成的「複雜網絡（complex network）」作為抽象模型來刻畫其行為。這些系統的關鍵在於——微觀的個體互動經由非線性機制，會湧現出宏觀層次的整體結構與功能。

隨著資料科學與計算能力的提升，複雜網絡分析（complex network analysis）逐漸成為探索各領域集體行為的重要方法之一。從早期以節點度數（degree）與最短路徑（shortest path）衡量網絡結構，到現今強調群聚度（clustering coefficient）、模組性（modularity）與橋接性（bridging），研究者得以從拓樸特徵中解析出潛藏於龐大資料背後的結構關係與動態機制。

在眾多網絡性質中，「連結（link）」是最基礎且最具代表性的元素。每一條邊代表兩個節點之間的互動關係，其強弱不僅影響資訊的傳播效率，更形塑整體網絡的穩定性與韌性。若將節點視為社會個體、伺服器或生物分子，則邊的強度便象徵它們之間的互賴程度。強連結（strong tie）通常存在於緊密群體中，代表頻繁互動與高信任；弱連結（weak tie）則橫跨群體邊界，雖互動稀疏卻能帶來關鍵的連結與新資訊。

Granovetter（1973）於其經典研究〈The Strength of Weak Ties〉中首次提出「弱連結的力量」理論，指出弱連結在社會網絡中具有跨越社群的橋梁效應。透過弱連結，資訊得以從不同群體間流動，促進創新與機會的擴散；而強連結則保障群體內部的穩定與合作。這一理論不僅奠定了社會網絡分析的基礎，也啟發後續大量研究，延伸至生態網、交通網、經濟網等多種應用場域。

然而，在面對龐大且異質的網絡結構時，單純以「強／弱」二分法描述連結強度已顯不足。實際上，連結的影響力常隨層級變化而不同，可能在局部區域具有關鍵性，但在全域尺度上影響有限。為了更精確描述這種層級性差異，黃崇源博士（2019）提出「階層邊類型分析（Hierarchical Edge Type Analysis, HETA）」演算法，透過共同鄰居比率（Common Neighbor Ratio, CNR）與統計閾值（threshold）來定義邊的多層次特徵。

HETA 的核心精神在於：連結的強弱並非固定，而是取決於其在網絡層次中的位置與可替代性。若兩端節點擁有大量共同鄰居，該邊即屬於黏著型（bond link）；若僅有少數共同鄰居，則可能是局部橋接（local bridge）或全域橋接（global bridge）；若某節點僅連接單一對象，其邊則被視為孤立連結（silk link）。這樣的層級分類方式讓研究者能從微觀互動推演宏觀結構，理解不同連結如何共同維持整體網絡的運作。

然而，原始 HETA 模型假設網絡整體密度分佈相對均勻。當應用至具有明顯密度差異的真實網絡時，例如城市交通網（核心密度高、郊區稀疏）、生物蛋白互作網（功能模組差異大）或社會關係網（圈層結構明顯），固定閾值將導致分類偏誤，使演算法無法真實反映邊在不同環境下的重要性。

為了克服此限制，本研究提出 Local-Density HETA（LD-HETA），在 HETA 的架構下導入局部密度自適應機制，使分類閾值能依邊所處區域的群聚程度動態調整。此方法兼具全域一致性與局部敏感度，並能更清晰揭示邊從「局部強」轉為「全域弱」的臨界點 β，提升異質拓樸網絡的分類精度與結構可解釋性。

\section{研究問題}

複雜網絡的重要議題之一在於如何正確刻畫邊的強度與結構影響力。傳統研究多採用全域一致的判定準則，假設網絡內整體密度大致均勻，但真實世界網絡往往呈現高度異質的區域結構，使全域閾值容易造成誤判。

以 HETA 為例，其固定閾值設計會造成以下問題：

\begin{enumerate}
    \item \textbf{高密度區域的過度黏著化（Over-bonding Bias）：}  
    高密度區域共同鄰居比例原本就高，導致大量邊被判為 bond link，使群內邊型缺乏分辨度。

    \item \textbf{低密度區域的橋接誤判（Bridge Overestimation）：}  
    稀疏區域共同鄰居比例自然偏低，即使該邊在結構上極為關鍵，也容易被誤分類為 global bridge。

    \item \textbf{階層不連續性（Hierarchical Inconsistency）：}  
    固定閾值會使邊在不同層次間的判斷不連續，缺乏拓樸一致性。
\end{enumerate}

近年研究（Li et al., 2021；Wang et al., 2022；Zhao et al., 2023）指出局部密度與群聚係數在網絡形成中扮演關鍵角色，若能讓分類閾值隨區域密度動態調整，可有效提升分類準確度並降低異質性偏誤。本研究因此提出將「局部密度自適應閾值」整合入 HETA 的改進方向。

\section{研究目標}

本研究旨在建立一套能兼顧階層性與密度敏感性的複雜網絡邊分類方法——Local-Density HETA（LD-HETA）。主要研究目標如下：

\begin{enumerate}
    \item 設計局部密度調整函數 $f(\rho_{ij})$，用以動態修正 HETA 的外部閾值 $TE^k$ 與內部閾值 $TI^k$。
    \item 延續 HETA 的層級架構，保留多層鄰居比率 $R^k_{i,j}$ 的遞迴定義，使 LD-HETA 具備可比較性與理論延續性。
    \item 定義密度偏移因子 $\Delta_k = \alpha(\rho_{ij}-\bar{\rho})$，控制密度異質性對分類的影響幅度。
    \item 在 WS、NW 模型與多模組真實網絡上驗證 LD-HETA 的穩定性與分類表現。
    \item 分析邊的轉折行為 β，探討其與局部密度的關聯。
\end{enumerate}

\section{研究貢獻與應用展望}

\subsection*{理論貢獻}

\begin{itemize}
    \item 建立層次性與密度敏感性兼具的邊分類模型。
    \item 提出自適應閾值函數機制，形式化描述閾值與局部密度的關係。
    \item 增強拓樸可解釋性，揭示邊由 bond 轉為 bridge 的臨界行為。
\end{itemize}

\subsection*{應用貢獻}

\begin{itemize}
    \item \textbf{社會網絡：}提升弱連結橋梁與群內黏著邊的辨識精度。
    \item \textbf{交通網絡：}識別高負載道路與關鍵連接邊，支持城市規劃。
    \item \textbf{生物網絡：}辨識模組間關鍵交互作用邊，協助生物功能分析。
    \item \textbf{通訊與安全：}發現維繫網絡韌性的敏感邊。
\end{itemize}

\section*{章節安排說明}

本論文的架構安排如下：

\begin{itemize}
    \item 第二章介紹複雜網絡與階層邊類型分析（HETA）之相關文獻，並說明局部密度在網絡分析中的重要性。
    \item 第三章提出本研究方法 Local-Density HETA（LD-HETA），包含自適應閾值設計、密度調整模型與階層推論架構。
    \item 第四章說明實驗設計與資料集，並比較 HETA 與 LD-HETA 在合成與真實網絡中的表現。
    \item 第第五章根據實驗結果進行分析與討論，特別是邊類型轉折 β 的行為與指紋穩定性。
    \item 第六章總結本研究並提出未來延伸方向，包括加權網絡、動態網絡與 GNN 整合等可能研究路徑。
\end{itemize}
